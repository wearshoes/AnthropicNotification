## Why

项目已部署并跑通，但 README 只有英文且内容简略。需要提供中英双语 README，默认显示中文，包含完整的功能介绍和 fork 后的配置/运行教程，方便中文用户快速上手。

## What Changes

- 重写 `README.md` 为中文版，包含功能介绍、架构说明、fork 配置教程、本地开发指南
- 新增 `README_EN.md` 英文版
- `README.md` 顶部添加语言切换链接

## Capabilities
